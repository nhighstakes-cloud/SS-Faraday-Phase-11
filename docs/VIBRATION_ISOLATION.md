# Vibration Isolation Strategy

## ## 🛡️ The Two-Layer Shield
1. **Passive Layer:** Pneumatic Optical Table + External Sorbothane Pucks (External).
2. **Vacuum Layer:** Viton (FKM) Flange Gaskets (Internal - ISO-K).
3. **Active Layer (Optional):** Minus-K MK52 Negative-Stiffness Platform.

## ## 📉 Transmissibility Logic
* **Floor Input:** 1.0 μm @ 963 Hz
* **Residual to Plates:** 0.208 nm
* **Target:** Suppression to < 0.1 nm via Active Damping.
