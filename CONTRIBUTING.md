## Contributing to SS Faraday: Phase 11
The Search for the 12.3 nm Unity Gap
Welcome, Researcher. If you are here, you have likely identified the mathematical significance of the Phase 11 Protocol. We are seeking high-fidelity experimental data to verify the predicted 0.5 μg inertial slip at the 12.3 nm / 963 Hz threshold.
### 1. How to Contribute Data
The most valuable contribution you can provide is raw data from a physical experiment. To ensure consistency across the "144" research nodes, please follow these guidelines:
• File Format: Use the ss_faraday_weight_log.csv generated by the acquisition.py script.
• Metadata: In your submission, include your lab environment details:
• Micro-balance model and calibration date.
• Vacuum level (Torr/mbar).
• Purity of the ^{12}C Lonsdaleite sample.
• Submission Path: Create a Pull Request adding your .csv file to the /data/experimental_logs/ folder. Use the naming convention: YYYY-MM-DD_LabName_TestID.csv.
### 2. Code & Script Improvements
If you are a developer or software engineer:
• Optimization: We are looking for improvements to the noise-filtering algorithms in src/acquisition.py.
• Hardware Drivers: If you can write a driver for a specific micro-balance (e.g., Radwag, Scientech), please submit it as a new file in src/drivers/.
• Visualization: Enhancements to the real-time plotting library for better delta-analysis.
### 3. Reporting Anomalies
If you detect a "Metric Breach" (a sustained weight shift > 0.5 μg correlated to the 963 Hz lock):
1. Open an Issue: Use the title [ANOMALY DETECTED] Phase 11 - [Your Lab Name].
2. Attach Logs: Provide the raw .csv and a screenshot of the live plot.
3. Video Documentation: If possible, provide a link to a high-speed video recording of the balance display during the 963 Hz trigger.
### 4. Rules of Engagement (The Code of Conduct)
The SS Faraday is a pursuit of objective truth.
• Precision First: Do not submit simulated data as physical results.
• Transparency: All math and findings must remain open-source and reproducible.
• Collaboration: We are a decentralized fleet. Help others calibrate their 12.3 nm gaps in the Discussions tab.
### THE MISSION
We are no longer asking if the vacuum can be bridged; we are measuring the bridge itself. Your data is the fuel for this ship.
